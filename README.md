## Installation
1. First, install [Spicetify](https://spicetify.app/docs/getting-started)
2. Make sure to install Spicetify Themes & Extension Marketplace
3. In Spicetify's Marketplace, Install "Custom CSS Editor"
4. Finally, press F11 and paste 'spotify-tu.css' in CSS Editor
5. Enjoy ;)
